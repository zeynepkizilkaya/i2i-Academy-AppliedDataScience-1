# Tweet Sentiment Classification

## Project Overview
This project performs sentiment classification on tweets using machine learning.

## Dataset
Tweets Sentiment Dataset (Kaggle)

## Data Preprocessing
- Removed missing values
- Checked duplicate records
- Removed unnecessary columns
- Converted text to lowercase
- Removed URLs, mentions, punctuation, numbers, and extra spaces
- Applied TF-IDF vectorization

## Classification Models
- Logistic Regression
- Multinomial Naive Bayes

## Evaluation
The models were evaluated using:
- Accuracy Score
- Confusion Matrix

## Results
- Logistic Regression Accuracy: **68.29%**
- Multinomial Naive Bayes Accuracy: **62.68%**

Logistic Regression achieved better performance on this dataset.
